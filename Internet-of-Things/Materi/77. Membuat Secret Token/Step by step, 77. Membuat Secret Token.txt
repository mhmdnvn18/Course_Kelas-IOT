Untuk membuat Secret Token/Menambahkan User:
1. Menekan icon gear pada domain yang sudah kita buat (namaInstance.cloud.shiftr.io)
2. Tekan tokens, lalu create token.
3. token ini digunakan agar user dapat terhubung ke shifter.io kita
4. Isi description (Bebas), lalu isi Secret (Password: random), Permision: Full Accses=Bisa publish/subscribe, Read-only=hanya Subscribe
5. jika ingin membuat user baru, bisa menekan create token lagi


Contoh:
mqtt://kelasiot76:JVxbR1CjvQJbblrJ@kelasiot76.cloud.shiftr.io

Username	:kelasiot76
Secret		:JVxbR1CjvQJbblrJ
Alamat Broker	:kelasiot76.cloud.shiftr.io

To connect to your instance, use as the host kelasiot76.cloud.shiftr.io and the MQTT port 1883. The username is always kelasiot76 while the password is the above configured token secret.
